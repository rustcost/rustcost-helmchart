1. Get the database connection string:
   If bundled Postgres enabled (default), the DATABASE_URL will be:
     postgres://<user>:<password>@postgres.<namespace>.svc.cluster.local:5432/<database>

   Example:
     postgres://rustcost:rustcostpassword@postgres.rustcost.svc.cluster.local:5432/rustcost

2. To use an external database:
   helm install rustcost ./rustcost-chart --set postgres.enable=false --set externalDatabase.enabled=true --set externalDatabase.url="postgres://id:pwd@host:port/dbname"

3. To override credentials:
   helm install rustcost ./rustcost-chart --set postgres.username=foo --set postgres.password=bar --set postgres.database=mydb
